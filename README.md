# Great Grandfather Bakery

A simple, static website for Great Grandfather Bakery built using Next.js.  
It showcases the bakery's menu, services, and contact information. Designed to be responsive and SEO-friendly.


Steps to run
```bash
git clone hhttps://github.com/omarakamal/great-grandfather-bakery.git
cd great-grandfather-bakery
npm i
npm run dev
```


## Tech Stack
- Next.js (React framework)
- JavaScript (ES6+)
- CSS Modules for styling
